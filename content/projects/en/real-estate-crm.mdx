## Overview

A web-based CRM application built for real estate agents. Covers client management across four roles (Buyer, Seller, Tenant, Landlord), property listings, appointment tracking, and sales records — surfaced through a central dashboard.

## Problem

Real estate agents typically track clients, properties, and appointments across spreadsheets and notes. Activity history gets lost, follow-ups are missed, and there's no central view of where each deal stands.

## Approach

Built on Next.js App Router with TypeScript. PostgreSQL handles persistence via Prisma ORM. The data model captures client roles, associated properties, scheduled appointments, and completed sales. The dashboard aggregates key figures across active deals.

## Outcome

In-progress project with the data model and core routing in place. Features are being added iteratively.
